GATKpipeline_germlineSV DRAFT_README
Pipeline for germline variant discovery.

I build this pipeline for germline variant discovery in pooled fungal samples using the "bootstrap proocol" in absence of a "true.vcf" (a set of known variants in the standard variant calling format ). 
This script needs to be integrated with 2 scripts that I used for vcfs filtration in the DP field after and before the bootstrap protocol steps. Those scripts uses a percentilic trimming approach to determ the DP thresholds for hard filtering using BAM files metrics. I will upload them soon!
