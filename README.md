Pipeline for germline variant discovery in pooled samples
I build this pipeline for germline variant discovery in pooled fungal samples using the "bootstrap proocol" in absence of a "true.vcf" 
(a set of known variants in the standard variant calling format ). 

This script needs to be integrated with an R script that I used for vcfs filtration in the DP field. That script uses a percentilic trimming approach 
to determ the DP thresholds for hard filtering using BAM files metrics. I will upload them soon!
